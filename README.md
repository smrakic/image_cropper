# Image Cropper Tool
Full-stack aplikacija za upload i kropovanje slika koja omogućava logo-overlay

# Tehnologije
1. Backend: .NET 10 Web API
2. Frontend: React
3. Kontejnerizacija: Docker

# Pokretanje aplikacije
Potreban je Docker Desktop
1. Klonirajte projekat:
   git clone https://github.com/smrakic/image_cropper.git
   cd image_cropper
2. Pokrenite Docker Desktop na svom uređaju
3. Pokrenite komandom
   docker-compose up --build
4. Link za pristup
   http://localhost:3000
